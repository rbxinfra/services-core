# Services Core

This repository contains all core code for services (primarily data access and business logic)

This repository also contains a legacy assembly: Roblox.ApiV2. Please refrain from using this as it is kept
for backwards compatibility for services wishing to step into Phase2 of BEDEV2 migration.
